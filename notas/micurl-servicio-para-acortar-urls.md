<p class="intro">Acortador de URLs propio para <a href="http://youare.com">YouAre</a>.</p>

Hace 48 horas que estrenamos <a href="http://micurl.com">micURL</a> (micro URL). Desarrollado por <a href="http://blogsmedia.com/equipo/gabriel.php">Gabriel</a>, es el servicio predeterminado para <acronym title="con id's permanentes">comprimir URLs</acronym> en <a href="http://youare.com">YouAre</a>. No queríamos depender de terceros.

¿Qué utilidad tienen este tipo de servicios? Entre ellas: haces URLs más amigables y memorizables en foros, sms, en los rótulos de televisión, que cuando insertes un enlace en un email evitar que el remitente se lo encuentre roto...

Entre las opciones contempla acortar las URLs desde la casilla de búsqueda de <a href="http://www.mozilla-europe.org/es/products/firefox/">Firefox</a> y "Copy to clipboard", opción que vimos tanto en <a href="http://urltea.com">urlTea</a> como en <a href="http://fon.gs">Fon.gs</a>, que reduce el proceso de copiar/pegar. <span style="text-decoration: line-through;">En las próximas semanas liberaremos su API</span> Ya hemos liberado la <a href="http://micurl.com/api-howto.php">API</a>. La intención es implantarlo de forma estándar a otras aplicaciones como plataforma de promoción de YouAre.

¿Te apetece descubrir <a href="http://es.wikipedia.org/wiki/Huevo_de_pascua_(virtual)">huevos de pascua</a> escribiendo <em>keywords</em> curiosas? <a href="http://micurl.com/war">http://micurl.com/war</a>, <a href="http://micurl.com/hillary">http://micurl.com/hillary</a>...
