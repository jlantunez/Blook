<p>Leo en el <a href="http://www.infonomia.com/blog/comentaris_post.asp?id=4941">blog de Infonomía</a> un post que reflexiona sobre los tan <a href="http://rvr.typepad.co/linotipo/2004/08/comunidades_de_.html">necesitados</a> y <a href="http://armengol.typepad.co/endocitosis/2004/08/blogs_temticos_.html">escasos</a> blogs de profesionales. Hablan de <a href="http://www.injef.com">INJEF.com</a>, que ofrece <a href="http://www.injef.com/php/index.php?option=content&amp;task=section&amp;id=1&amp;Itemid=194">espacio gratuito</a> para la creación de bitácoras destinadas a profesionales del sector jurídico, fiscal y empresarial:</p>

<blockquote><b>Blog colectivo profesional: derecho, empresa y fiscalidad</b>
20/08/2004 / <a href="http://injef.com/php/index.php?option=com_mamblog&amp;task=show&amp;action=user&amp;id=117&amp;Itemid=194">Luis Sotillos</a> <br /><br />INJEF.com acaba de inaugurar hace unas semanas su <a href="http://injef.com/php/index.php?option=com_mamblog&amp;Itemid=167">blog colectivo profesional</a> sobre temas jurídicos, empresariales y de Internet. Cada autor escribe y mantiene su propio weblog, pero todos juntos contribuyen a un único blog colectivo: este puede leerse bajo el enlace "últimas anotaciones" de las bitácoras profesionales de INJEF.com.</blockquote>
