Un cliente de Blogs Media busca a personas que les apasione viajar para cubrir puestos de redactores web y un <em>community manager</em>. <a href="http://blogsmedia.com/contacto">¿Te interesa?</a>

<h2>Requisitos para optar a redactor web</h2>

¿Te gusta viajar? ¿Cuentas anécdotas simpáticas sobre los lugares que visitas? ¿La gente te pregunta sobre los sitios de moda de las ciudades que has recorrido? ¿Tienes un blog? ¿Subes vídeos a <a href="http://youtube.com">YouTube</a>? ¿Tienes una cuenta de <a href="http://flickr.com">Flickr</a>?

Tienes la oportunidad de combinar tu pasión y divertirte mientras haces tu trabajo. Se está preparando un proyecto en el que se pretende enriquecer la experiencia de viaje de la gente que visite la plataforma web.

<h2>Community manager</h2>

Si además de todas las características mencionadas, también eres una persona ordenada, con criterio y con experiencia en la gestión de equipos, tanto a nivel presencial como a distancia, puede que seas el <em>Community Manager</em> que estamos buscando.

Para ambos puestos, se requiere de excelente ortografía y un alto nivel de inglés. Si además eres autónomo o no te importaría serlo, preferible.

¿Das el perfil? <a href="http://blogsmedia.com/contacto">Envíanos tus datos</a> y hablamos.
